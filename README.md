这是一个心理健康支持 agent 原型项目，包含项目内 Codex skills、安全规则和合成测试用例。

第一阶段聚焦焦虑支持。项目故意先从安全门开始，再逐步加入焦虑相关工作流。

## 项目结构

- `.agents/skills/`：仅在本项目内使用的 Codex skills。
- `safety/`：共享的临床边界、安全分流和响应规则。
- `evals/`：合成安全测试和回归测试用例。
- `docs/`：设计说明和资料来源摘要。

## 当前 Skills

- `mental-health-safety-harness`：所有心理健康支持流程的上层安全门。
- `risk-crisis-triage`：危机、自伤、自杀、伤人、精神病性症状、严重功能受损、药物和医疗红旗分流。

## 非临床定位

本项目用于症状缓解、心理教育、自助指导、筛查辅助和转介支持。它不是诊断系统、治疗系统、药物建议系统或危机服务。

内部分类只用于选择更安全的支持路径，不应作为诊断或疾病标签展示给用户。

## 语言约定

- `README.md` 默认中文。
- `README.en.md` 提供英文版本。
- 除 README 系列外，项目文档、规则、skills、evals 和治理文件默认使用英文。
- 最终产品回复应使用用户的偏好语言。
