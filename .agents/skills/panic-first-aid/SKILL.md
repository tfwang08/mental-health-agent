# Panic First Aid

## Purpose

Provide short, practical support for a sudden anxiety or panic-like surge.

This skill is not a diagnostic tool. It helps the user stabilize in the moment after safety and medical-red-flag checks have already been completed.

## Clinical Basis

Based on:

- NICE CG113 stepped-care principles.
- Low-intensity CBT-style psychoeducation and self-help support.
- Project evidence policy in `../../../docs/evidence-policy.md`.
- Project medical red-flag guidance in `../../../safety/medical-red-flags.md`.
- Project clinical boundaries in `../../../safety/clinical-boundaries.md`.

## Allowed Behaviors

The skill may:

- Acknowledge that panic-like surges can feel intense and frightening.
- Explain that strong body sensations can rise and fall without labeling the experience as a disorder.
- Guide one short grounding or breathing step.
- Help the user reduce avoidance of immediate safe surroundings.
- Suggest tracking what happened after the surge has settled.
- Recommend professional evaluation when episodes are repeated, impairing, new, worsening, or unclear.

## Not Allowed

The skill must not:

- Diagnose panic disorder or any other condition.
- Say the symptoms are definitely anxiety.
- Reassure the user that there is no medical issue.
- Replace medical evaluation when warning signs are present.
- Provide medication advice.
- Run long analysis while the user is highly distressed.

## Preconditions

Before using this skill:

1. Apply `mental-health-safety-harness`.
2. If the selected mode is not ordinary support, follow the selected mode instead.
3. If medical warning signs are present, stop this workflow and follow medical redirect guidance.

## Response Pattern

Keep the first response brief:

1. Name the immediate goal: help the body settle for the next few minutes.
2. State that this is not a diagnosis.
3. Offer one simple exercise.
4. Ask one practical check-in question.

Example structure:

- "Let's focus on getting through the next few minutes. This is not a diagnosis."
- "Put both feet on the floor and name five things you can see."
- "Now slow the exhale slightly for three breaths."
- "Is the intensity changing at all, even a little?"

## Exercise Options

Use only one at a time.

### Grounding

Ask the user to notice:

- five things they can see;
- four things they can feel;
- three things they can hear;
- two things they can smell;
- one thing they can taste or one neutral object they can describe.

### Breathing Support

Use gentle pacing only:

- breathe in normally;
- make the exhale slightly longer;
- repeat for three to five breaths.

Do not force deep breathing or long breath holds.

### Orientation

Ask the user to name:

- where they are;
- what day or time it is;
- one safe or neutral thing in the room;
- one next small action.

## Follow-Up After Stabilization

After the user is calmer, offer to summarize:

- what was happening before the surge;
- body sensations noticed;
- what helped even slightly;
- whether this has happened before;
- whether professional evaluation may be useful.

## Handoff

- If warning signs appear, use medical redirect guidance.
- If the pattern is repeated, worsening, or impairing, recommend professional evaluation.
- If ordinary support remains appropriate, continue with `anxiety-intake-assessment`, worry sorting, monitoring, or referral summary skills.
