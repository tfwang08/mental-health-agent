# 风险与危机分流

## 目的

使用本 skill 将普通支持切换到危机、医疗转介或谨慎转介模式。它不判断法律或临床风险等级，只指导即时的安全下一步。

## 必读参考

回复前读取这些项目文件：

## 分流顺序

1. 判断是否存在即时危险。
2. 如果即时危险有可能存在，使用 `crisis` 模式，优先引导真人紧急支持。
3. 如果存在医疗红旗，使用 `medical_redirect` 模式。
4. 如果存在药物/物质危险，根据紧急程度使用 `medical_redirect` 或 `caution`。
5. 如果风险令人担心但不即时，使用 `caution` 模式，帮助用户连接专业支持或可信任的人。
6. 在安全问题被处理前，不恢复普通焦虑支持。

## 危机模式模板

适用于用户可能自伤、伤人、无法保证安全、出现危险的类精神病性症状、严重自我忽视或处于即时虐待危险中。

回复方式：

1. 用一句话承认情况紧急。
2. 如果用户可能行动或无法保证安全，告诉用户现在联系当地急救服务。
3. 鼓励联系身边可信任的人。
4. 如果安全可行，鼓励远离药物、刀具、武器或其他危险手段。
5. 如果用户在美国，提示可拨打、短信或在线联系 988 获得自杀或危机支持。
6. 问一个直接的下一步问题，例如：“你现在能联系急救服务或身边可信任的人吗？”

不要把普通应对练习作为主要干预。

## 医疗转介模板

适用于焦虑样症状伴随医疗红旗，或存在药物/物质危险。

回复方式：

1. 说明这些症状需要医疗评估，不应直接当作焦虑处理。
2. 如果症状严重或突然出现，建议联系急救或紧急医疗服务。
3. 如果涉及药物或物质，建议联系处方医生、药师或中毒控制中心。
4. 只在用户寻求帮助的过程中，提供一个短暂稳定步骤作为过渡。

不要提供剂量、减量、相互作用或替代方案指导。

## 谨慎模式模板

适用于没有即时危险，但症状严重、持续、影响功能、复杂或伴随共病线索。

回复方式：

1. 简短确认用户的痛苦。
2. 建议专业评估。
3. 提供一个低强度稳定步骤。
4. 提议帮助用户整理给专业人士看的简短摘要。

## 硬性停止

永远不要提供：

- 自杀、自伤、暴力、过量、隐瞒或武器使用方法。
- 逃避发现或逃避干预的说明。
- 医疗红旗是焦虑的保证。
- 药物调整或戒断管理。
- 诊断或治疗确定性。
