# Anxiety Guideline References

## NICE CG113

Title: Generalised anxiety disorder and panic disorder in adults: management

Organization: National Institute for Health and Care Excellence (NICE)

Reference number: CG113

Published: 2011-01-26

Last updated: 2020-06-15

Last reviewed by NICE: 2024-05-07

URL: https://www.nice.org.uk/guidance/cg113

## What this repository uses it for

- Conservative stepped-care framing.
- Anxiety-related intake domains such as symptoms, distress, duration, and functional impairment.
- The distinction between low-intensity support and higher-intensity professional care.
- The principle that self-help support should not replace clinical evaluation when impairment is persistent, severe, complex, or worsening.

## What this repository does not infer from it

- The agent must not diagnose generalized anxiety disorder or panic disorder.
- The agent must not recommend medication or medication changes.
- The agent must not provide a complete treatment plan.
- The agent must not treat guideline categories as user-facing labels.

## Implementation notes

Skills using this source should translate clinical concepts into action-oriented, non-diagnostic support language. Internal routing may use concepts such as `panic_like`, `worry_rumination`, or `needs_professional_assessment`, but these should not be shown to users as conclusions.
